## One-Time Path Adjustment

This project uses the `/10/` URL prefix by default. All `.html` and `.css` paths will be updated to match this prefix.

### Step 1 — Run the script

From the project root, run:

    python .py

This command updates all `.html` and `.css` files in place.

### Step 2 — Delete the script

`.py` is a one-time utility. After execution, delete it if it’s not required for other development purposes.
